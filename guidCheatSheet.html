Cheat Sheet

A quick reference for Form.io GUIDs and their meanings, and other miscellaneous information

VBScript

For use in Server Side Form Script

TheForm.User.OwnerObjectGUID.toString()

The ContactGUID of the contact that "owns" the form instance. This describes the open contact.

TheForm.User.ContactGUID.toString()

The ContactGUID of the signed in user. This describes the contact of the user that is accessing the form.

TheForm.User.UserGUID.toString()

The UserGUID of the currently signed in user. This describes the user the is accessing the form.

TheForm.FormDefinition.FormDefinitionGUID.toString()

The FormDefinitionGUID of the current form. This tells the system which form is being referenced.

TheForm.FormInstanceGUID.toString()

The FormInstanceGUID of the current form. This tells the system which instance of the current form is being referenced.

theForm.GetGuid().toString()

This generates a new GUID.
