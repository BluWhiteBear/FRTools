Change Log

Dated list of changes and improvements made to the site and it's tools

Printout From Form | Version 0.4.2

October 24, 2025

New Features

  • Added "Help" tab that gives tool usage instructions and additional information

Improvements

  • Improved code comments and documentation
  • Fixed an issue where the "Reset to Defaults" button was throwing an error due to a setting being removed previously
  • Fixed an issue where radio button panels were scaled as if they had an extra option inside them
  • Added "Apply Settings" button to Settings tab so user doesn't have to reupload their form to apply new settings
  • Label Height is now variable
  • Field Height is now variable
  • Added Snap Grid Size property to generated report

Printout From Form | Version 0.4.1

October 23, 2025

New Features

  • Added "Settings" tab that allows the user to define certain layout properties for the report
  • Report Width is now variable
  • Margin Sizes are now variable
  • Vertical Component Spacing is now variable
  • Font properties for Report Header are now variable
  • Font properties for Report Section are now variable
  • Font properties for Field Labels are now variable
  • Font properties for Field Outputs are now variable

Improvements

  • labelPosition values "right-left" and "right-right" now properly position their field label
  • Form and Data Grid DetailReport Bands now generate within the report layout instead of being appended at the end of the layout

Printout From Form | Version 0.4.0

October 17, 2025

Improvements

  • Improved code comments and documentation
  • Column Element weights now respect their component width property
  • Radio Button Elements now respects their "Inline" layout property, dynamically adjusting the option widths as well
  • Text Field, Text Area, and Select Elements now respect their "labelPosition" property, dynamically adjusting their field label positions
  • Fixed an issue that was causing some Markup tags to not close properly
  • Fixed an issue that was causing the "Color" Markup tag to not render correctly
  • Fixed an issue that was causing the labels of nested Panel elements to inaccurately size horizontally
  • Panel Labels now have a unique style defined in devexpress-definitions.js
  • Elements now have the full width of the report available for their layout instead of being capped at a width of 650px
  • Form and Data Grids now generate inside their own DetailReport Bands

Printout From Form | Version 0.3.8

Improvements

  • Improved code comments and documentation
  • Printouts are now generated with hidden and private properties set to true
  • Content components now properly calculate their height based on the rendered content
  • HTML Element and Content components now map their HTML tag content to DevExpress MarkUp tags

Printout From Form | Version 0.3.7

October 7, 2025

Improvements

  • Improved code comments and documentation
  • Fixed an issue that caused Panel labels to generate with inaccurate/repeating ItemNums, causing other elements to fail DevExpress parsing
  • XML Validation now does a repair pass, attempting to correct any structural issues regarding ItemNumbers
  • Fixed an issue that caused container elements to not scale accurately when nested with empty containers

Notes

  • NestedForms have been temporarily disabled as they are causing issues with XML generation
